$Page MIDIを介してインストゥルメントを再生する
$TargetName Playing the Instrument via MIDI

ソフトウェア・インストゥルメントを再生するためには、まず、インストゥルメントのトラックが「アーム」されていることを確認してください。 通常、インストゥルメントが挿入されると自動的にアームされますが、トラックミキサーの「アーム」ボタンをクリックして、手動でアームすることもできます。
ArmButtonInst.tif

トラックをアームすると、以下の2つが行えるようになります。

-> トラックが外部MIDIキーボードからのMIDIノートを受信できるようになります（外部MIDIキーボードがない場合、ご使用のコンピュータ・キーボードを使用できます。これについては追って説明します）。
-> トラックが録音可能になり、再生内容をいつでも録音することができるようになります。

MIDIキーボードがご使用のコンピュータに接続されている場合、キーボードを使ってノートを再生してみましょう。インストゥルメントが聞こえ、トラックビューのインストゥルメントに含まれている「入力メーター」と「出力メーター」が反応しているのが分かります。
TrackViewMeters.tif

$Comment -----------------------------------------------------------------------
$Page コンピュータ・キーボードを使用する
$TargetName Using the Computer Keyboard

MIDIキーボードをお持ちでない場合は、ご使用のコンピュータ・キーボードで代用できます。コンピュータ・キーボードを使用するには、スクリーン右上隅の「コンピュータMIDIキーボード」スイッチがオンになっていることを確認してください。
ComputerKeyboardSwitchInst.tif

-> キーボード上列の文字キー（QWER...）は、ピアノの黒鍵を再生します。
-> 中央列の文字キー（ASDF...）は、白鍵を再生します。
-> ［Y］キーと［X］キーは、コンピュータMIDIキーボードの幅をトランスポーズ（移調）します。
-> ［C］キーと［V］キーは、再生するノートのベロシティ（強度）を変化させます。

アプリケーション・ウィンドウ下部の「ステータスバー」には、トランスポーズ・キーまたはベロシティ・キーを押した際にその効果が表示されます。
StatusBarOctave.tif


Divider.tif
メモ：ご使用のコンピュータ・キーボードのキーは、$0内の任意のコントロールにマップすることもできます。詳しくは、マニュアルの「MIDIとキーリモートコントロール」の章をご参照ください。

ご使用のコンピュータのキーボードからのI/Oはすべて、$0ではなく、コンピュータのオペレーティング・システムにより調節されています。サウンドを試聴する場合、コンピュータのMIDIキーボードで問題ありませんが、さらに正確でキー感触が重要となる入力を行う必要がある場合、外部キーボードまたはコントロールサーフェスを使用することをお勧めします。

$Page プリセットをホットスワップする
$TargetName Hot-Swapping Presets

$0のライブラリには多種多様なサウンドが収録されています。別のプリセットをブラウズしたいときは、デバイスのタイトルバーの「ホットスワップ」ボタンをクリックします。
BrowseButtonInst.tif

最後に選択したプリセットがブラウザ内にハイライト表示されます。プリセット名の右には、ホットスワップボタンが付いています。
BrowseButtonTwinInst.tif

このホットスワップボタンを押すと、ブラウザとトラックビューのデバイスの間に一時的なリンクが作成されます。プリセットを選択してからコンピュータ・キーボードのEnter/Returnキーを押すと、前回ロードしたインストゥルメントと新しいプリセットが置き換わります。プリセットが別のインストゥルメント（「Impulse」など）に対するものである場合、$0により自動的にデバイスが交換されます。

適切なインストゥルメントが見つかったら、ご使用のコンピュータ・キーボードの［Esc］キーを押し、ホットスワップ・モードを終了します。

$0ブラウザには、プリセットをキーワードから探すことができる便利な検索機能も付属しています。たとえば、「ブラウザ検索」ボタンをクリックしてから「bass」をタイプ入力し、「Return」キーを押すと、$0のライブラリ内のベースのプリセットすべてがリスト表示されます。
BrowserSearchButton.tif

$Page 外部プラグインを使う
$TargetName Using External Plug-Ins

$0の内蔵インストゥルメントとエフェクトに加え、VSTまたはAudio Units（Macのみ）形式のサードパーティ製プラグインを使用することもできます。プラグインの使い方は、内蔵デバイスの使い方とよく似ています。プラグイン・インストゥルメントを挿入するには、まず、このセレクタを使って「プラグインブラウザ」を開きます。
PluginBrowserSelector.tif

メモ：プラグインを始めて使用する場合は、$0のプラグイン対応をオンにする必要があることもあります。プラグインブラウザの「有効」ボタンを押すと、ファイル/フォルダ設定が表示されます。ファイル/フォルダ設定では、プラグインを設定することができます。

プラグインはすべてブラウザにリスト表示され、形式、制作者の順で分類されています。インストゥルメント・プラグインには、ブラウザ・アイコンの下半分に小さな「ピアノキー」が付いており、エフェクト・プラグインと異なって表示されます。
PluginBrowserInst.tif

$0のデバイス同様、インストゥルメントを選択し、MIDIトラックにドラッグします。プラグインの「カスタムエディタウィンドウ」が自動的に開きます。トラックビューには、アサイン可能な「X-Yグリッド」と共にプラグイン・デバイスが表示されます。
PlugInTrackView.tif

$Page メトロノーム
$TargetName The Metronome

このレッスンを終了する前に、$0のマルチトラックテープレコーダである「アレンジメント」に演奏内容を録音する方法について簡単に見ておきましょう。

すべてが同期で再生されるようにするためには、同期で録音することが非常に重要になります。同期で録音する最も簡単な方法は、$0内蔵の「メトロノーム」を使用することです。メトロノームは、コントロールバー・スイッチからオンにできます。スイッチは、アプリケーション・ウィンドウ左上隅にあります。
MetronomeSwitch.tif

$0が再生中に、メトロノームがテンポを刻みます。「スペースキー」を押して再生を開始してから、マスタートラックの「キューボリューム」コントロールを使ってメトロノームのボリュームを調節します。
CueVolume.tif

停止するには、スペースキーをもう一度押します。

$Page 録音と再生
$TargetName Recording and Playback

次のステップでは、録音過程を開始します。これまでの段階でアームボタンをオンにしており、$0がMIDIノートに反応するようになっていますので、MIDIトラックは録音可能になっています。
ArrangerTracksInst.tif

コントロールバーの「停止」ボタンを2回押し、アレンジメント位置を「1.1.1」にリセットします。
TransportStopRec.tif

次に、となりの「グローバル録音」ボタンを押します。（マルチトラックテープレコーダを使ったことがあれば、同様の操作なので分かりやすいかもしれません。各トラックをアームし録音可能にしていれば、グローバル録音ボタンを使ってそれらのトラックを録音可能に切り替えることができます。）
GlobalRecordRec.tif

最後に、コントロールバーの「再生」ボタンを押し（または、ご使用のコンピュータ・キーボードの「スペースキー」を押し）、録音を開始します。
TransportPlayRec.tif

録音が進むと同時に、新しい「MIDIクリップ」がトラック内に作成されていきます。
ArrangementMIDIClipRecording.tif

録音を停止するには、停止ボタンを押します。停止ボタンをもう一度押してアレンジメントを「1.1.1」へリセットしてから、再生ボタンを押して演奏内容を聞きます。

$Page クリップの詳細を見る

新規に録音されたクリップを使って、様々なことが行えます。

$Page ノートの編集
$TargetName Editing Notes

MIDIエディタには、ビートタイムルーラーが付いています。ビートタイムルーラーは、アレンジメント同様、水平方向にズームすることができます。ピアノキーの左をドラッグし、エディタを垂直方向にズームやスクロールすることもできます。
MIDIEditorScrollAreas.tif

コントロールバーの「ドローモード」スイッチをオフにすると、マウスを使ってノートを選択して移動することができます。ノート周辺の選択ボックスをドラッグして、ノートをまとめて動かすことができます。
DrawModeOffInst.tif

各ノートには、MIDIエディタ下に「ベロシティマーカー」がついています。このマーカーは、ノートベロシティ（ノートが再生される際の強度）を示しています。ベロシティは、ドラッグして変更することができます。
VelocityEditorInst.tif
