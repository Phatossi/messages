$Page オートメーションを使用する
$TargetName Using Automation

$0では、ミキサーやエフェクト内部のコントロールの動きを「オートメーション化」することができます。面白い効果を生み出すので、ダイナミックな音楽を制作するのに便利です。このレッスンでは、オートメーションの作成・記録・編集を行う方法を説明しています。

Divider.tif

レッスンを進める前に、ブラウザの表示/非表示ボタンを押してスクリーン右手のブラウザを閉じ、アレンジメントビューの表示部分を広げましょう。下の画像のボタンがそれです。
ShowHideBrowser.tif

画面右側のミキサーセクションを使って、必要なセクション（「ミキサーセクション」）のみが表示され、その他が非表示になるよう設定すると良いでしょう。
MixerSectionShow2.tif

準備が整ったら、下の「次のページ」をクリックして進みます。

$Page オートメーション
$TargetName Automation

オートメーションとは、ソングの時間軸上で生じるノブ・スイッチ・コントロールの動きをいいます。$0では、オートメーションを記録・編集することができます。

「ビートルーラ」を上方向にドラッグして、アレンジメントビューをズームアウトしてください。下の画像がビートルーラです。
TimeRuler.tif

トラック｢Bass｣を見てみましょう。記録したオートメーションの好例を見ることができます。 小節33をズーム表示すると見やすくなります。
ExampleOfAutomation.tif

クリップの波形、またはMIDIノートにブレークポイントエンベロープが重ねられると、オートメーションがアレンジメントビューに表示されます。ここで紹介するオートメーションはトラックパニングです。トラック名称のすぐ下にある2つの「オートメーションセレクタ」に表示されています。
UnderTheName.tif

$Page オートメーションをドローする
$TargetName Drawing Automation

コントロールバースイッチのドローモードがオンの時、$0のエンベロープをドローすることができます：
ThisSwitch.tif

ドローモードがオンのとき、クリップの波形やMIDIノート上に直接、下の画像のようなステップをドローすることができます。
StepsOnto.tif

ステップのサイズは、編集グリッド幅により異なります。編集グリッド幅は、「オプション」メニュー、またはアレンジメント内のクリップタイトルバー以外の部分で$ContextMenuを使いコンテクストメニューを表示させ、変更することができます。
GridContextMenuArr.tif

ステップではなく、「フリーハンド」でグリッドをドローする場合には、グリッドをオフに切り換えます。

Divider.tif
オートメーションの結果は、クリップのタイトルバーをクリックしてからスペースバーを押すことでいつでも確認することができます。こうすると、クリップの最初から再生が開始されます。スペースバーをもう一度押すと再生が停止します。

$TargetName Creating New Envelopes

セレクタから、トラックのミキサーまたはデバイスのパラメータを選択し、エンベロープをゼロからドローすることができます。
UnderTheName.tif

$0のミキサーとデバイスコントロールにクリックするだけで、オートメーション・エンベロープを呼び出し編集することができます。 トラック｢Drums｣をダブルクリックし、トラックのDrum RackインストゥルメントとBeat Repeatエフェクトをスクリーン下部に表示します。

ここで、Beat Repeatのコントロールのいずれかをクリックすると、Beat Repeatのオートメーション・エンベロープがセレクタとトラック表示内に表示されます。
NewEnvelope.tif

$Page オートメーションを編集する
$TargetName Editing Automation

ドローモードスイッチがオフのとき、オートメーションエンベロープはブレークポイントを表示しています。
Breakpoint.tif

トラック「Bass」のオートメーションを使って、ブレークポイントを新しいポジションへドラッグしてみて下さい。

エンベロープ上をダブルクリックして、ブレークポイントを作成したり削除したりすることができます。

オートメーションを編集する場合、オートメーションが適用されているトラックパラメータだけを表示すると便利です。これは、いずれかのオートメーションセレクタから「オートメーションされたパラメータのみ表示」を選択して行います。
ShowAutomated.tif

このモードでは、各トラックの2つのオートメーションセレクタが1つになり、「オートメーションされた」パラメータだけが表示されます。

この編集モードを解除するには、セレクタから「すべてのパラメータを表示」を選択します。

$Page オートメーションレーン
$TargetName Automation Lanes

オートメーション・セレクタ下の｢+｣ボタンを押して、現在のパラメータ・エンベロープを｢オートメーションレーン｣内に表示することができます。
MakeNewLane.tif

その後、オートメーション・セレクタからさらにパラメータを選択することができます。必要に応じて、レーンを作成することもできます。 こうすることで、複数のパラメータを同時に表示し編集することができます。
MultipleLanes.tif

ドローモードがオフの場合、バックグラウンドで1つまたは複数のレーンをクリック&ドラッグし、時間を選択することができます。
SelectedEnvelope.tif

その後、｢複製｣などの｢編集｣メニューのコマンドを使うと、エンベロープにのみ影響し、クリップの内容自体には影響しません。
DuplicatedEnvelope.tif

$Page オートメーションの記録
$TargetName Recording Automation

オートメーションを使って、デバイスコントロールにリアルタイムで加えられた変更内容を記録することができます。

$0のエフェクトとインストゥルメントは、デバイスブラウザに置かれています。デバイスブラウザは、スクリーンの左側にあり、セレクタボタンをクリックすると開きます。
SelectorClicked.tif

デバイスブラウザの左側にあるスクロールバーを使って、「Audio Effects」フォルダ内の「Auto Filter」エフェクトまでスクロールして下さい。
AutoFilter.tif

Auto Filterを右へクリック&ドラッグし、エフェクトをトラック｢Bass｣へドロップします。 スクリーン下部の｢トラックビュー｣に表示されます。
EffectInTrack.tif

$Comment -----------------------------------------------------------------------
$Page オートメーションの記録（続き）
$TargetName Recording Automation, Continued

これで、Auto Filterのコントロールに加えられる変更をアレンジメントの一部として記録できます。

これを行うには、まず、コントロールバーの「停止ボタン」を2度押し、アレンジメント位置を1.1.1.にリセットします。
StopButton2.tif

その後、コントロールバーの録音ボタンを押します。
RecordButton2.tif

コントロールバーの再生ボタンを押して、記録を開始します。
PlayButton.tif

Auto Filterのコントロール（周波数コントロールなど）を動かすと、$0は変更の正確なログを記録します。
Frequency.tif

スペースバーを一度押すと記録が終了します。もう一度スペースバーを押すと、変更内容を聴くことができます。Auto Filterのコントロールがオートメーション化され、自動で動いている様子を見ることができます。

記録されたコントロールの動きは、ブレークポイントエンベロープとしてトラック「Bass」内に表示されます。このエンベロープをさらに編集することもできます。
BassAutomation.tif
