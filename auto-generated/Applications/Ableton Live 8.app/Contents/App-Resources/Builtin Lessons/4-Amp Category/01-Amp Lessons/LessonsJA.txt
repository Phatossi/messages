/Ampへようこそ！/

*Amp*は、温かみとドライブを簡単に加えることのできるAbletonオーディオエフェクトです。 Softubeとの共同開発により制作されたAmpは、伝説のアンプとキャビネットのサウンドと簡潔さを忠実に再現します。

Ampに付属するCabinetは別個のエフェクトで、最適化されたマイクおよびマイク位置のギターキャビネットのセレクションを提供します。 また、AbletonのインストゥルメントとエフェクトをAmpの新しい音色特性で変化させるサウンドの数々も収録しました。 これには、インストゥルメントラック、エフェクトラック、Liveクリップ、400を越えるプリセットが含まれます。

Ampを使用したギタープリセットにすばやくアクセスするには、こちらのリンクを使用します。

-> $Link *Guitar Amp Clean*<BrowseLibraryPresets:/Audio Effects/Audio Effect Rack/Guitar Amp Clean/> オーディオエフェクトプリセット
-> $Link *Guitar Amp Distortion*<BrowseLibraryPresets:/Audio Effects/Audio Effect Rack/Guitar Amp Distortion/> オーディオエフェクトプリセット
-> $Link *Guitar Processing*<BrowseLibraryPresets:/Audio Effects/Audio Effect Rack/Guitar Processing/> オーディオエフェクトプリセット
-> $Link *Bass Amp*<BrowseLibraryPresets:/Audio Effects/Audio Effect Rack/Bass Amp/> オーディオエフェクトプリセット

AmpとCabinetデバイスを使用したオーディオエフェクトラックとインストゥルメントラックもいくつか用意されています。

7種類のアンプ・モデルから選択できます。

-> [*Clean*]は、60年代のクラシック・アンプの「ブリリアント」チャンネルをベースとしています。 このアンプは、ブリティッシュ・インヴェイジョンを代表するギタリストに広く使用されました。

-> [*Boost*]は、同じアンプの「トレモロ」チャンネルをベースとしており、エッジの効いたロックのリフに最適です。

-> [*Blues*]は、ブライト感のある70年代のギターアンプをベースとしています。 このクラシック・アンプは、カントリー、ロック、ブルースのギタリストに人気が高いアンプです。

-> [*Rock*]は、60年代の45ワットのクラシック・アンプをモデルに作成されています。 最も有名なロック・アンプといえるかもしれません。

-> [*Lead*]は、メタルのギタリストに人気の高いハイゲイン・アンプの「モダン」チャンネルをベースとしています。

-> [*Heavy*]は、同じアンプの「ヴィンテージ」チャンネルをベースとしており、メタルやグランジのサウンドにも最適です。

-> [*Bass*]は、力強いローエンドとファジーな高音域でベーシストに高い人気を博した70年代の珍しいPAをモデルに作成されています。
