$Page Affectations instantanées
$TargetName Instant Mappings

Des affectations instantanées sont disponibles pour de nombreux contrôleurs MIDI. Ces contrôleurs pris en charge de façon native ont en commun l'approche de l'interfaçage, même quand les caractéristiques prises en charge varient en raison des limitations matérielles.

-> $Link *Contrôle de périphérique* <GotoPage:Device Control>*:* Les paramètres d'instruments et d'effets peuvent être contrôlés via votre contrôleur MIDI. Cette fonctionnalité vous évite le cauchemar du réglage de chaque paramètre à la souris.

-> $Link *Contrôle du mélangeur* <GotoPage:Mixer Control>*:* Les contrôleurs ayant une configuration appropriée de faders, commandes rotatives et boutons peuvent gérer le mélangeur de $0.

-> $Link *Contrôle du transport* <GotoPage:Transport Control>*:* Certaines surfaces de contrôle peuvent piloter les fonctions de transport de $0.

-> $Link *Contrôle des pads* <GotoPage:Pad Control>*:*  Certaines surfaces de contrôle ayant des pads peuvent automatiquement s'actualiser pour contrôler le jeu de pads visible dans le Rack de batterie ou /Drum Rack/ de Live.

$Comment -----------------------------------------------------------------------
$Page Contrôle de périphérique
$TargetName Device Control

Le contrôle du son d'un instrument ou d'un effet peut être primordial pour votre travail. Toutefois, affiner des paramètres à la souris peut être fastidieux. A titre d'antidote, Ableton a développé les affectations instantanées qui donnent un accès facile aux paramètres de périphériques grâce à des surfaces de contrôle.

La plupart des surfaces de contrôle prises en charge de façon native disposent d'au moins un jeu de huit commandes rotatives ou encodeurs. Ces commandes sont instantanément associées aux paramètres du périphérique actuellement sélectionné, quel qu'il soit.

Divider.tif

Pour permettre un accès à plus de huit paramètres, nous avons divisé les paramètres de chaque périphérique en banques, les paramètres liés étant regroupés dans la même banque (par ex. les paramètres de filtre). Comme de nombreux contrôleurs actuels ne disposent pas d'afficheurs graphiques indiquant les paramètres affectés, ceci aide à améliorer votre flux de production.

Le changement de banque se fait à l'aide de deux boutons (banque précédente/suivante) ou de huit (un par banque), selon la configuration matérielle du contrôleur. Certains contrôleurs n'ont pas assez de boutons pour le changement de banque. Dans ce cas, nous avons prévu un florilège ou banque "Best-of" pour chaque instrument et effet.

$Page Contrôle du mélangeur
$TargetName Mixer Control

Réaliser un mixage propre est essentiel en production musicale, mais là encore, la souris est un outil inadéquat pour gérer dynamiquement les signaux des pistes. Les affectations instantanées disponibles pour le mélangeur de $0 améliorent grandement la situation.

Les contrôleurs MIDI ayant des faders appropriés peuvent régler les niveaux de multiples pistes audio et MIDI simultanément.
Sliders.tif

Si un contrôleur dispose d'un fader supplémentaire dédié, il sera affecté au /volume Master/ du mélangeur.

Les contrôleurs MIDI ayant un jeu de boutons supplémentaire les utiliseront pour armer les pistes en vue d'enregistrement. Presser un bouton d'armement sélectionnera aussi sa piste et tout instrument qu'elle contient.

Tout jeu supplémentaire de commandes rotatives, d'encodeurs ou de curseurs est affecté aux /valeurs de départ (Send)/ de piste.

$Page Contrôle du transport
$TargetName Transport Control

Si vous aimez jouer conjointement à vos morceaux ou enregistrer de nouvelles pistes durant la lecture, vous avez probablement constaté qu'il n'est pas très pratique de devoir utiliser la souris pour lancer et arrêter l'enregistrement et la lecture. Certains contrôleurs sont équipés de boutons de transport qui résolvent ce problème:
Transport.tif

Les affectations instantanées pour les boutons de transport couvrent les fonctions suivantes:

-> *Lecture:* Lance ou relance la lecture.
-> *Stop:* Stoppe la lecture ou ramène au début du morceau si la lecture est déjà stoppée.
-> *Avance rapide:* Fait avancer la position dans le morceau d'une mesure (et continue ainsi si le bouton est gardé enfoncé).
-> *Retour rapide:* Fait reculer la position dans le morceau d'une mesure (et continue ainsi si le bouton est gardé enfoncé).
-> *Enregistrement:* Commute le bouton /Enregistrement global/ de $0.
-> *Boucle:* Commute le bouton /Boucle/ de $0.

FaderfoxesG1.tif
Pour utiliser les contrôleurs LV1 et LX1 via le support natif dans Live, veuillez tenir compte des réglages qui se trouvent au chapitre "Réglages en mode système" des modes d'emploi des contrôleurs. Le choix correct de la configuration interne est particulièrement important - le LV1 utilise la configuration 2.

Il n'est plus nécessaire d'utiliser les Sets Live fournis contenant les différentes affectations MIDI; le support natif dans Live fournira automatiquement des affectations instantanées.

De plus, un seul câble est nécessaire entre le port de sortie du contrôleur et le port d'entrée de votre interface MIDI pour établir la connexion; une connexion de retour vers le contrôleur n'est pas nécessaire.

Voici une liste des fonctions de contrôleur qui diffèrent des descriptions du mode d'emploi des contrôleurs ou qui s'y ajoutent:

-> Fonctions de lancement de scène et de clip automatiquement assignées pour les 8 premières scènes et les 8 premiers clips de chaque piste. Note: le lancement de clip n'est disponible qu'en mode de déclenchement (Trigger).

-> Contrôle de la banque "Best-of" de paramètres pour le périphérique sélectionné via les encodeurs dans les groupes de potentiomètres "FX3" et "FX4".

-> Vous pouvez contrôler les paramètres du dernier périphérique EQ-Eight dans la piste sélectionnée via le groupe de piste alternatif en pressing Shift et les boutons Pot. Le périphérique n'a pas besoin d'être sélectionné.

-> Le joystick contrôle les paramètres X/Y du périphérique sélectionné dans les groupes Pot "FX1" et "FX2".

-> Changez de fenêtre (c-à-d Arrangement/Session ou Piste/Clip) avec les deux boutons blancs du LV1 en maintenant le bouton Shift.

Vous pouvez bien sûr changer toutes les assignations en mode d'apprentissage MIDI de Live.
