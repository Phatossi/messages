$Page ベースパッチを作成する
$TargetName Creating a Bass Patch
$ListInTocForProduct 40

Operatorの音響可能性について理解が深まったところで、ここではお好みにあったベースサウンドの作成方法について学びましょう。Operatorによる音源へのハイブリッドアプローチにより、アナログスタイルのサウンドから少し変わったメタリックなサウンドまで、さまざまな種類のサウンドを作成することができます。

Divider.tif
*メモ:* このレッスンでは、Liveセットを使用し、実際に操作を行いながら学ぶことができます。 $Link ここ <Set:/Sets/OP-Default Set.als> をクリックして、Liveセットをロードしましょう。
Divider.tif

他のOperatorレッスン同様、「トラックビュー」にインストゥルメントを表示すると分かりやすくなります。 スクリーン左下隅の表示/非表示ボタンをクリックして「インフォビュー」を閉じましょう。
HideInfoBass.tif

Divider.tif
ヒント：このチュートリアルではベースサウンドを作成しますので、キーボードの低オクターブ部分を使用するとよいでしょう。小型のキーボードを使用する場合は、キーを1，2オクターブ下にトランスポーズし、正しいサウンドレンジになるよう調整します。

コンピュータのキーボードを使用する場合、ZとXのキーを使ってトランスポーズできます。このレッスンには、C2からC3が適当です。
Divider.tif

$Comment --------------------------------------------------------------------------------
$Page アルゴリズムを選択する
$TargetName Choosing an Algorithm

Operatorでは、4つのオシレータのFM変調ルーティング設定にアルゴリズムを使用します。アルゴリズムにより、一つ一つ特長と利点が異なります。例えば、最初の8つのアルゴリズムはFM構成に適しているのに対し、その他の3つのアルゴリズムはハイブリッドまたはアナログスタイルの減算パッチに適しています。

アルゴリズムにアクセスするには、シェルのグローバルセクションを選択し反転表示させます。下の画像がシェルのグローバルセクションです。
GlobalSectionBass.tif
アルゴリズムは、Operatorディスプレイの最上部に表示されます。

デフォルトでは、Operatorのアルゴリズムは1番最初のアルゴリズムが選択されています。このアルゴリズムでは、オシレータDがCを、CがBを、BがAをモジュレートします。
Algorithms.tif
この設定では、オシレータAのレベルを変更して全体のボリュームをコントロールします。他のオシレータのレベルを変更するとFMアマウントがコントロールされ、サウンドのティンバー（音色）を調整できます。

$Comment --------------------------------------------------------------------------------
$Page チューニングとレベル
$TargetName Tuning and Levels

オシレータBはオシレータAをモジュレートしています。再生しモジュレーションの効果を確かめながら、オシレータBのレベルをゆっくり-6dBまで上げてみましょう。
OscBLevelBass.tif
レベル値が上がるに従って、音が変化していくのが分かるはずです。レベルが高いほど、サウンドに多くの倍音がかかります。

ここで、オシレータBのコースチューニングを調整します。このコントロールで設定するチューニング値によってそれぞれ全く異なる倍音が生まれ、音調の幅が広がります。通常、設定値が高いほど、さらにハードでブライトになります。

少し試してみたところで、コースチューニング値を0.5に、レベルを-20dBに設定しましょう。
OscBLevelAndCoarse.tif
低めのオクターブを再生すると、エレキベースを彷彿とさせる表現力豊かなベース音が聞こえるはずです。

ヒント：場合によっては、マウスを使って設定するよりも、Operatorコントロールをクリックし数値を入力する方が便利です。

$Comment --------------------------------------------------------------------------------
$Page FM変調を加える
$TargetName Adding FM Modulation

それではここで、他のオシレータを使って音質をさらに高めてみましょう。

オシレータCはオシレータBにFM変調を加えます。オシレータCのレベルとチューニングをいろいろ変えてみましょう。コースチューニング値を高めに設定すると、デジタル感がさらに深まります。（チューニングに加える変化の結果を聞くには、オシレータCのレベルを上げる必要があることを忘れないで下さい。）

いろいろ試したら、コース値を12に、レベルを-30dBに設定しましょう：
OscCLevelBass.tif

$Comment --------------------------------------------------------------------------------
$Page エンベロープを使用する
$TargetName Using Envelopes

オシレータCのエンベロープを使って、サウンドの音色に動きを与えてみましょう。エンベロープは、オシレータのサウンド変調レベルを時間の流れに沿って変化させます。

オシレータCがシェルから選択されていることを確認して、サステインレベルを-21dBに設定します：
Sustain-21.tif
ディケイが短いと、サウンドのアタックにわずかにとぎれが生じます。ディケイが長いと、ノートが保たれている間、倍音がゆっくりとフェードします。ディケイパラメータを操作して、違いを確認してみて下さい。

ディケイタイムを397msに設定しましょう：
Decay397.tif

$Comment --------------------------------------------------------------------------------
$Page ベロシティを使って強弱をつける
$TargetName Using Velocity to Create Dynamics

サウンドボリュームをノートベロシティに対応させ、再生内容に強弱をつけることができます。

シェルからクリックしてオシレータAに戻り...
OscABass2.tif
...レベル<ベロシティパラメータを下の画像のように75%に設定します：
VelocityLevel.tif
これで、サウンドが再生内容に対して動的に反応するようになります。

$Comment --------------------------------------------------------------------------------
$Page 不協和要素を加える
$TargetName Adding Inharmonic Elements

ここまでの過程で、多様な音楽ジャンルに合う典型的なFMベースサウンドを作成することができました。ここでは、このベースサウンドを全く異なるサウンドへと変形させてみましょう。

オシレータBのコースチューニングを1に、ファインチューニングを500に変更します。こうすることで、サウンド特性がメタリックに変化し、不協和要素が加わります。
OscBCoarseFine.tif
ここで、レベルを-7dBまで上げ、オシレータの効果が良く聞こえるようにします：
IncreaseLevel.tif
オシレータCのオンとオフを切り換え、どう変化するか実験してみてもいいでしょう。切り換えるには、シェルのCアイコンをクリックします。下の画像がアイコンです。
CIcon.tif
オシレータCがオシレータBをモジュレートしているかどうかにより、サウンド構造が変化するのが分かるはずです。

A・B・Dのアイコンをクリックして、Operatorの他のオシレータのオンとオフを同様に切り換えることができます。

次のページへ進む前に、4つのオシレータが全て有効になっていることを確認して下さい。

$Comment --------------------------------------------------------------------------------
$Page ボリュームとエンベロープコントロール
$TargetName Volume and Envelope Control

メタリックなサウンドを、オシレータBのエンベロープを使ってパーカッシブに変化させることができます。

まず最初に、オシレータBがシェルから選択されていることを確認します：
OscBBass.tif
ディケイを1.44sに、サステインを-infdBに設定します：
MetallicOscB.tif
このパッチは、再生中に歪みやすく変化していますので、グローバルボリュームコントロールを使ってサウンドのボリュームを-20dBまで下げましょう：
Volume20.tif

$Comment --------------------------------------------------------------------------------
$Page フィルタセクション
$TargetName The Filter Section

ベースサウンドをいくつか作成し、FM合成の仕組みが分かったところで、今度はフィルタを加えてみましょう。

まず始めに、フィルタセクションを有効にします。下の画像のように、"Filter（フィルタ）"と書かれた所の下にある四角ボタンをクリックします：
FilterOnBass.tif

フィルタを有効にしたら、フィルタ周波数ノブを上下にドラッグして調節します。その後、500Hzに戻します。
AFreqyKnob.tif

$Comment --------------------------------------------------------------------------------
$Page カットオフとレゾナンス
$TargetName Cutoff and Resonance

レゾナンスノブを調節して、フィルタの動作をさらに調整することができます。

レゾナンスノブを上下にドラッグしてみましょう。その後、7.00に設定します：
ResTo7.tif
レゾナンス値が高いとカットオフ周波数が強調され、サウンドにエレクトロニック感が加わります。

フィルタ周波数ノブを上下にドラッグし、高レゾナンス設定がカットオフ周波数にどのように影響するか試してみましょう。試し終わったら、フィルタ周波数ノブを1.9kHzに設定します：
Cutoff19.tif
フィルタセクションのオンとオフを切り換え、フィルタがあるかないかでサウンドがどのように違うかを比べることができます。
FilterOff.tif
フィルタを再びオンに切り換え、フィルタ周波数<ベロシティ（Freq<Vel）パラメータを75%に設定し、レスポンスにダイナミックさを加えます（このエフェクトを聞くには、ベロシティセンシティブ・キーボードが必要です）：
DynamicFilter.tif

$Page 最後に
$TargetName ... and Finally

柔軟なFM変調ルーティングシステムと一般的なフィルタを組み合わせることにより、Operatorでは、ベースサウンド（ベースサウンドに限らずあらゆるサウンド）作成に対する合成方法の選択肢が多様に揃っています。次のレッスンでは、アナログドラムマシーンサウンド作成をOperatorで行う方法について見ていきます。

インフォビューには、Operatorのパラメータに関する説明が表示され大変便利です。インフォビューの表示と非表示を切り換えるには、スクリーン下左隅の三角ボタンをクリックします。
BassInfoView.tif
