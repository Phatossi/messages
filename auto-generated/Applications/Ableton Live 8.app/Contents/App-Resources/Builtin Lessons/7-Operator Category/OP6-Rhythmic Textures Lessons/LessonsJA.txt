$Page リズム構成を作成する
$TargetName Creating Rhythmic Textures
$ListInTocForProduct 40

シリーズの最終レッスンでは、これまで学んできたスキルを全て使用します。このレッスンでは、Operator独自のエンベロープのループ機能に焦点を当て、他のシンセサイザーでは作成の困難な、難解なリズム構成を作成する方法について説明していきます。

このレッスン内で説明される概念を理解しておくためにも、これより前のレッスンにまだ目を通していない場合は、先へ進む前に目を通しておくと良いでしょう。

Divider.tif
*メモ:* このレッスンでは、Liveセットを使用し、実際に操作を行いながら学ぶことができます。 $Link ここ <Set:/Sets/OP-Default Set.als> をクリックして、Liveセットをロードしましょう。
Divider.tif

他のOperatorレッスン同様、スクリーン左下隅の表示/非表示ボタンをクリックしてインフォビューを閉じ、トラックビュー内のインストゥルメント表示スペースを広げることができます。
HideInfoViewFx.tif

$Comment --------------------------------------------------------------------------------
$Page オシレータを設定する
$TargetName Setting up the Oscillators

オシレータのルーティングを設定しましょう。

シェルからOperatorのグローバルセクションをクリックして、ディスプレイに表示させます。
GlobalSection.tif
その後、2つのオシレータが積み重なっているアルゴリズムを選択します。
Algorithm.tif

このアルゴリズムを使えば、FM・アナログ合成両方の利点を活用することができます。

オシレータAのレベルを-3.5dBに設定します。チューニングパラメータはどれもデフォルト値のままにしておきます。
OscALevel.tif

$Comment --------------------------------------------------------------------------------
$Page エンベロープを使いリズムを加える
$TargetName Using Envelopes to Add Rhythm

それでは、パッチにリズムを加えていきましょう。

オシレータAのエンベロープのアタックを0.38msに変更し、少しソフトにします。そして、ディケイを9.90セカンドに延ばし、サステインを-infdBに短縮します：
RhythmAtkDec.tif

エンベロープモードセクションから、LoopをSyncに切り換えます：
RhythmEnvLoop.tif

ここで、リピート（ビート/同期レート）を1/4に設定します：
RhythmRepeat.tif

オシレータAのサイン波パルスのボリュームが4分音符になります。

$Comment --------------------------------------------------------------------------------
$Page FM変調を加える
$TargetName Adding FM Modulation

オシレータBに切り替えましょう。オシレータBは、オシレータAをFMを介してモジュレートするよう設定されています。 チューニング・コントロールは、デフォルト設定のままにしておきます。こうすることで、2つのオシレータのチューニングの比率が1:1になり、ノコギリ波のようなFM音色になります。

ここで、レベルを-20 dBに上げましょう。
OscBLevel.tif

$Comment --------------------------------------------------------------------------------
$Page オシレータBとループエンベロープ
$TargetName Oscillator B + Looped Envelopes

オシレータBのエンベロープを使い、オシレータAのリズムをさらに複雑にしてみましょう。

ディケイを452msに、サステインを-42dBに設定します：
OscBEnv.tif

エンベロープモードをSyncに設定します：
RhythmEnvLoop2.tif

リピート周波数を1/8に設定します：
RhythmRepeat2.tif

オシレータAの4分の1音符リズムに、繊細な8分の1音符リズムが加えられました。

ハイブリッドな構成を持つFM合成要素が完成するまでの過程の、折り返し地点までたどり着くことができました。オシレータBのコースチューニングを変更してみると、音色が大きく変化するのが分かるはずです。

$Comment --------------------------------------------------------------------------------
$Page オシレータCとループエンベロープ
$TargetName Oscillator C + Looped Envelopes

オシレータCを使って、2分音符のリズムを持つソフトなうねりを加えることができます。

まず最初に、効果が聞こえるようオシレータCのレベルを-3.5dBに上げます：
OscCRhythm.tif

ディスプレイから、オシレータCの波形をSaw 3（Sw3）に変更し、音色に繊細さを加えます。
OscCSaw.tif

ここで、ディスプレイを再びエンベロープに切り替えましょう。 アタックを306 msに、ディケイを2.11セカンドに、サステインを-inf dBに設定し、わずかにフェードイン・フェードアウトするようにします。
OscCEnvelopes.tif

他のオシレータと同様、エンベロープをLoopからSyncに設定し、リピートタイムを1/2に設定します。
RhythmRepeat3.tif

$Comment --------------------------------------------------------------------------------
$Page オシレータDとループエンベロープ
$TargetName Oscillator D + Looped Envelopes

2分の1音符のうねりにブライトな16分音符パルスを加え、パッチに対するFM関連の処理を完成させましょう。

まず最初に、オシレータDのレベルを-0.0dBに設定し、オシレータDがオシレータCを最大の強度でモジュレートするようにします：
OscDLevel.tif

オシレータDのディケイを397msに、サステインを-infdBに短縮してサウンドをパーカッシブにし、モードセクションを設定値1/16のSyncに設定します。
OscDEnvelopes.tif

リズムのFM要素により、オシレータCのうねりにシャープでスタッカートな16分音符のパーコレーション（浸出部分）が生まれます。

オシレータDのコースとレベル設定を変化させてみると、サウンドの性質が劇的に変化しますので、いろいろと試してみるのもいいでしょう。試行錯誤がOperatorマスターへの一番の近道です。

$Comment --------------------------------------------------------------------------------
$Page サウンドのフィルタリング
$TargetName Filtering the Sound

リズムFM要素をそのままに、共振ローパスフィルタを加えサウンドを盛り上げましょう。

フィルタセクション左端の四角ボタンをクリックして、フィルタセクションを有効にします：
FilterOnRhythm.tif

フィルタタイプとフィルタ周波数（Freq）は、デフォルト値のままにしておきます。 [Low 12dB]を500 Hzに、レゾナンス（Res）を3.70まで上げて、アナログ・スタイルらしさを演出します。
FilterSettings.tif

$Comment --------------------------------------------------------------------------------
$Page フィルタエンベロープをループさせる
$TargetName Looping the Filter Envelope

Operatorのエンベロープは全てループさせることができます。フィルタエンベロープのリズムを作成し、複雑さを加えましょう。

ディケイを397msに、サステインを12%に設定します：
FilterEnvelopes.tif

ここで、Syncをオンにし、リピートを1/4に設定します。
FilterSync.tif

ここからは、効果が聞こえるよう、エンベロープ値を上げる必要があります。 値を96%に設定してみましょう。
EnvelopeAmount.tif

$Comment --------------------------------------------------------------------------------
$Page 仕上げを行う
$TargetName The Finishing Touches

サウンドの微調整を行うには、フィルタの共振パルスを調節しているフィルタエンベロープのアタック・ディケイ・リピートの設定をいろいろ変えてみます。

また、タイムとトーンを調節し、パッチをさらに精緻にすることもできます。
TimeTone.tif

フィルタ設定が完了したら、スプレッドコントロールを使ってサウンドにふくらみを与え、最後の仕上げを加えることができます。スプレッド値を最大に上げましょう。スプレッドは、その効果の程度にかかわらず、ボイスを2倍使用することに注意して下さい。
SpreadRhythm.tif

$Page 最後に
$TargetName ... and Finally

Operatorシリーズのレッスン全てを終えたら、Operatorマスターへの第一歩を踏み出したことになります。ベース・リード・パッド・ドラム・リズム構成をカスタマイズし使いこなすのはあなたです。Operatorで使用可能な合成ツールに関する詳細については、 $Link リファレンスマニュアル <Manual> の「Liveインストゥルメントリファレンス」の章をご参照ください。

インフォビューには、Operatorのパラメータに関する説明が表示され大変便利です。インフォビューの表示と非表示を切り換えるには、スクリーン下左隅の三角ボタンをクリックします。
FxInfoView.tif
