$Page $0 LE のインターフェース
$TargetName The $0 LE Interface

$0での音楽制作における構成単位となるものを、「クリップ」と呼びます。 クリップは、音楽素材の一部分です。 メロディ・ドラムパターン・ベースラインや楽曲などがクリップとなります。

$0で行う操作のほとんどが、クリップを作成・編集・組み合わせる作業です。こうして、さらに長い音楽を構築していきます。 これらの作業は、「アレンジメント」と「セッション」とよばれる2つのビューを使って行います。

セッションビューでは、クリップは以下のように表示されます。
SessionClips.tif

アレンジメントでは、クリップは以下のように表示されます。
ArrangementClips.tif

コンピュータの*Tab*キーまたはビューセレクタを使って、2つのビューを素早く切り替えることができます。
ViewSelectors.tif

$Page ブラウザとライブラリを開く
$TargetName Opening the Browser and Library

$0にデータを作成またはインポートする方法はたくさんありますが、「ブラウザ」からドラッグするのが一番簡単です。 ブラウザのセレクタボタンまたは$ViewPanMod-Bを使って、ブラウザ表示を切り替えることができます。
BrowserSelector.tif

ブラウザを使って、ハードディスクの任意のロケーションや$0の「ライブラリ」を表示することができます。ライブラリでは、$0にあらかじめインストールされているコンテンツを選択できます。 ライブラリを選択するには、セレクタを使って3つの「ファイルブラウザ」のうちの1つをを開きます。
FileBrowserSelectors.tif

その後、ブラウザの最上部のライブラリ「ブックマーク」を選択します。
LibraryBookmark.tif

$Page クリップを操作する
$TargetName Working With Clips

ライブラリの「Clips」フォルダには、あらかじめ作成されているクリップが含まれており、すぐさま使用できる状態になっています。 ここでは、MIDIデータ・インストゥルメント・エフェクトが1つにまとめられています。 それでは、そのうちの1つをロードしてみましょう。 まず、セッションビューに切り替えます。
SessionView.tif

その後、「Clips」フォルダを表示し、クリップを選択してセッションのからのスロットへドラッグします。
ClipToSlot.tif

ライブラリの「Samples」フォルダからオーディオファイルをドラッグし、新規クリップを自動作成することもできます。 MIDIまたはオーディオクリップでも同様の操作を行え、この2種類のクリップを同一のプロジェクト内に並べることができます。

クリップをロードしたら、再生ボタンを押してクリップを「ラウンチ」することができます。
PlayButtons.tif

クリップにはそれぞれ独自の再生ボタンがついているので、個別にラウンチすることができます。 クリップの横列を、「シーン」と呼びます。 シーン内に配置されたクリップは、マスタートラックの「シーンのラウンチ」ボタンを押してすべてを同時にラウンチすることができます。
SceneLaunch.tif

$Page $0 のコンテンツをインストールし使用する
$TargetName Installing and Using $0 Content

$0LEには、プログラムに慣れすぐさま創作活動を開始できるよう、いくつかのコンテンツがあらかじめインストールされています。 インストールされているコンテンツは以下の通りです。

-> 「Simpler」および「Impulse」インストゥルメント
-> レッスンおよびインストゥルメントと Live エフェクト・デバイスのプリセットを収録した「Liveパック」
-> $0LEの機能を耳で確認し、詳しく学ぶことのできる「デモソング」

$0LEは、自動的にインストールされないコンテンツも付属しています。これらのコンテンツは、必要に応じてインストールすることができます。 インストールされていないコンテンツは以下の通りです。

-> 「エッセンシャル・インストゥルメント・コレクションLEエディション」 高品質マルチサンプル・インストゥルメントを収録したこのパックは、Simplerインストゥルメントでバラエティ豊かにすぐさま使用できます。
-> 「Unnatural Selection」 $Link Puremagnetik <http://www.puremagnetik.com/> による、ループ・ビート・インストゥルメントプリセットのコレクションです。

Liveパックはいくつかの方法でインストールすることができますが、「ファイル」メニューから「Liveパックをインストール」コマンドを使用するのが一番簡単です。 通常のオペレーティング・システム・ウィンドウが開き、追加Liveパックが表示されます。

無料コンテンツに加え、$0LEには「Operator」および「Sampler」のプリセットをインストールするLiveパックも付属しています。 これらのインストゥルメントは $0LEに付属していませんが、 $Link Abletonウェブショップ <http://www.ableton.com/shop> から別途購入が可能です。 $Link 「デモモード」 <Lesson:Builtin/Demo Mode> を有効にし、これらのインストゥルメントを試用することもできます。 デモモードでは、アドオン製品の機能を試用することができますが、作業内容の保存と書き出しはできません。

Ableton では、くつろぎながら$0について学べる、機能別の$0説明ムービーも作成しています。 ムービーは、インストールディスクまたは $Link Abletonウェブサイト <http://www.ableton.com/movies> にてご覧いただけます。

（いくつかのムービーは、フルバージョンのLiveの機能について説明しています。そのため、LiveLEでは使用できない機能である場合がありますのでご注意ください。）
