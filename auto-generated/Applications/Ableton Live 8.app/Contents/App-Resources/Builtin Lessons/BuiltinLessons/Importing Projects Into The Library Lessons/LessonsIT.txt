$Page Esportazione dei Progetti nella Libreria
$TargetName Exporting Projects to the Library

Ogni Live Clip, campione o preset che memorizzi nella Libreria di Live è disponibile per qualsiasi Progetto. Puoi utilizzare il pulsante /Esporta/ del File Manager per copiare nella Libreria tutti i file dal Progetto che stai gestendo:
ExportSection.tif

Il tuo Progetto non sarà influenzato dal processo di esportazione, in quanto i suoi file vengono solo copiati, non spostati.


Divider.tif
*Nota:* se desideri eliminare un Progetto dopo l'esportazione, usa il Browser dei File di Live.
