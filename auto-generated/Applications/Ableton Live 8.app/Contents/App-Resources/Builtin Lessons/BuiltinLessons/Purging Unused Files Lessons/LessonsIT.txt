$Page Eliminazione dei File Inutilizzati
$TargetName Purging Unused Samples

Il File Manager di Live è in grado di trovare per te i file inutilizzati di un Progetto. Successivamente puoi quindi rivederli ed eventualmente cancellarli, singolarmente o tutti in una volta.

Durante la ricerca dei file "inutilizzati", Live ispezionerà ogni file (campioni, etc.) di una cartella di Progetto, verificando se appartiene ai riferimenti dei Live Set, delle Live Clip o dei preset dei dispositivi del Progetto. In caso contrario, il file è considerato inutilizzato - / anche se ancora utilizzato da altri Progetti o altri programmi ./

Per vedere i file inutilizzati del Progetto corrente, usa i pulsanti /Mostra/ di ciascuna categoria di file:
UnusedSamplesSection.tif

Live elencherà i file inutilizzati nel Browser dei File, dove, se vuoi, puoi ascoltarli in anteprima e/o eliminarli.
