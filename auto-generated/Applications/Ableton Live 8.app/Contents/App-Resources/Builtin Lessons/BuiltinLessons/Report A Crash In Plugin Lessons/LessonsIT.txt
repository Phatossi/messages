Apprezziamo che tu dedichi del tempo per segnalare un crash. Grazie per il tuo aiuto!

I dati di crash forniti dal sistema suggeriscono che il problema potrebbe essere correlato a questo CRASHED_PLUGIN_TYPE plug-in:

*CRASHED_PLUGIN_NAME (versione CRASHED_PLUGIN_VERSION)*

Verifica presso il rispettivo produttore che questo plug-in sia aggiornato. Accertati, inoltre, di stare utilizzando $Link l'ultima versione di Live <http://www.ableton.com/latest-versions>.

Divider.tif
Affinché possiamo analizzare ulteriormente il problema, clicca sul link sottostante per visualizzare il crash report Pack sul tuo computer:

CRASH_REPORTING_PACKAGE_NAME

Per cortesia, invia questo file a $Link crashes@ableton.com <mailto:crashes@ableton.com>.
Divider.tif

Questo pacchetto di segnalazione di crash contiene informazioni generate dall'ultima esecuzione di Live:

-> i file delle preferenze e dell'autorizzazione di Live
-> un registro di ciò che il software ha fatto durante l'ultima esecuzione del programma
-> tutti i Live Set, i preset e i dispositivi di Max usati durante l'ultima esecuzione del programma (tutti senza campioni)
-> vari altri file di registro (log) creati da Live
-> informazioni di crash del sistema operativo

Nessun'altra informazione viene inviata. Useremo i dati di questa segnalazione di malfunzionamenti solo per correggere i bug. Per sapere come utilizziamo i tuoi dati, leggi la nostra $Link politica sulla privacy <http://www.ableton.com/privacy-policy>.

*Nota:* riceverai una risposta automatica alla tua segnalazione di crash. Qualora avessi bisogno di ulteriore aiuto, sei pregato di contattare il nostro $Link supporto tecnico <http://www.ableton.com/support>.
