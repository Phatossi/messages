Nous vous sommes reconnaissants de prendre le temps de nous rapporter un crash. Merci pour votre aide!

Live a créé un Pack rapport de crash. Veuillez cliquer sur le lien ci-dessous pour faire apparaître ce fichier sur votre ordinateur:

CRASH_REPORTING_PACKAGE_NAME

Veuillez envoyer ce fichier à $Link crashes@ableton.com <mailto:crashes@ableton.com>.

*Note:* vous recevrez une réponse automatique à votre rapport de crash. S'il vous de l'aide, veuillez contacter notre $Link équipe d'assistance<http://www.ableton.com/support>.

Divider.tif

Ce Pack rapport de crash contient des informations qui ont été générées la dernière fois que vous avez fait fonctionner Live:

-> les fichiers de préférences et d'autorisation de Live
-> un journal de ce qu'a fait le logiciel durant la dernière exécution du programme
-> tous les Sets Live, presets et périphériques Max utilisés durant la dernière exécution du programme (tous sans échantillons)
-> divers autres fichiers-journaux créés par Live
-> les informations de plantage du système d'exploitation

Aucune autre information n'est envoyée. Nous n'utiliserons les données contenues dans ce rapport que pour corriger les bogues. Voir notre $Link politique de confidentialité <http://www.ableton.com/privacy-policy> pour découvrir comment nous utilisons vos informations.
