Vous pouvez archiver un dossier projet en créant un /Pack Live./ Cela copie le contenu de votre dossier projet dans un seul fichier compressé qui peut être facilement sauvegardé ou partagé.
PackingSection.tif

Comme le Pack Live ne fait que dupliquer le contenu de votre dossier projet, il ne comprendra pas les échantillons externes auxquels votre projet se réfère. Si vous avez besoin de créer une archive complète d'un projet, utilisez d'abord la section /Echantillons externes/ du gestionnaire de fichier pour réunir ces fichiers dans le dossier projet.

Les Packs Live utilisent une compression sans perte pour réduire leur taille. Selon les échantillons utilisés, la taille totale du fichier peut être réduite de jusqu'à 50 pour cent sans aucunement altérer vos données.

Vous pouvez décompacter les Packs Live en double-cliquant dessus, en les déposant dans $0, ou en utilisant la commande *Installer Live Pack* du menu Fichier.


Divider.tif
*Note:* Si vous voulez supprimer un projet après compactage, utilisez le navigateur de fichiers de $0.
