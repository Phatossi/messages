Die /APC20/ (Ableton Performance Controller) ist eine spezielle Hardware-Bedienoberfläche für Ableton Live, die vom gleichen Entwicklerteam stammt wie die legendäre Akai MPC-Produktserie. Dieser Kurs hilft Ihnen beim Einrichten Ihrer APC20 und macht Sie mit ihren Funktionen und Möglichkeiten vertraut.

Klicken Sie bitte $Link hier <Set:/Sets/APC20 Demo.als>, um ein Live-Set einzuladen, mit dem Sie arbeiten können, während Sie die einzelnen APC20-Funktionen kennenlernen. Klicken Sie dann auf /Nächste Seite/ unten rechts in diesem Kurs, um weiterzublättern oder benutzen Sie zur schnellen Navigation die folgenden Links:

Nachdem Sie Live freigeschaltet, die APC20 eingeschaltet und an den USB-Port Ihres Rechners angeschlossen haben, müssen Sie die APC20 in Lives /Voreinstellungen/ zuordnen und aktivieren.

1) Öffnen Sie Lives Voreinstellungen über das /Live-Menü/ unter OS X, oder das /Optionen-Menü/ unter Windows

2) Gehen Sie auf die /MIDI/Sync-Seite/ (mit Klick auf den gleichnamigen Reiter)

3) Wählen Sie unter /Bedienoberfläche/ das APC20-Preset  aus. Stellen Sie dann in den Menüs /Eingang/ und /Ausgang/ den MIDI-Port ein, an den Sie die APC20 angeschlossen haben:

$Page Die Clip-Launch-Matrix
$TargetName Clip Launch Matrix

Mit der Tasten-Matrix der APC20 können Sie direkt auf die Clips in Ihrer Session-Ansicht zugreifen. Die Matrix besteht aus 40 dreifarbigen Tasten in einer 5x8-Anordnung. Von der Matrix werden somit jeweils 5 Szenen und 8 Spuren pro Szene repräsentiert. Die Farbkodierung der Tasten informiert Sie genau über den Zustand der Clips und mit den Cursor-Tasten (Left-Right-Up-Down) der APC20 können Sie sehr einfach in Lives Session-Ansicht navigieren.
LaunchMatrix.tif

Das Drücken eines der 40 /Clip-Launch-Taster/ startet den Clip im entsprechenden Clip-Slot in Live. Wenn eine Spur aufnahmebereit geschaltet ist (Record Arm), wird mit dem Drücken der Taste ein neuer Clip aufgenommen.

Die Taster der Matrix leuchten in drei verschiedenen Farben, um den Zustand des Slots/Clips mitzuteilen:

-> Gelb: dieser Slot enthält einen Clip
-> Grün: dieser Clip wird abgespielt
-> Rot: dieser Clip wird aufgenommen
-> Keine Farbe: dieser Slot ist leer

Viele Aspekte von Lives Programmoberfläche werden auf der APC20 kopiert. Sie können z.B. die gesamte Musik einer Spur mit dem /Clip-Stop-Taster/ der Spur stoppen oder ganze horizontale Linien von Clips mit den /Scene-Launch-Tasten/ "abfeuern". Das Drücken von /Shift/ und eine der /Clip-Stop-Tasten/ stoppt die Wiedergabe aller Clips.

Die APC20-Bedienoberfläche informiert Sie darüber, was in der Live-Software passiert. Genauso wichtig ist jedoch auch, dass die Software den Zustand der Hardware-Bedienoberfläche reflektiert. Zum Beispiel werden die Clip-Slots, die momentan von der Tasten-Matrix gesteuert werden, in Live mit einer roten Umrandung angezeigt.
RedBorder.tif

Die Cursor-Steuerung ermöglicht Ihnen die Navigation in der Session-Ansicht:

-> Mit Up oder Down bewegen Sie sich jeweils um eine Szene nach oben oder unten.
-> Mit Left und Right bewegen Sie sich jeweils eine Spur nach links oder rechts weiter.

$Comment -----------------------------------------------------------------------
$Page Die Session-Übersicht
$TargetName Session Overview

Die /Session-Übersicht/ der APC20 lässt Sie schnell durch große Live-Sets navigieren, ohne auf den Bildschirm des Rechners schauen zu müssen. Wenn Sie die Shift-Taste halten, zoomt die Matrix auf eine Übersichtsdarstellung Ihrer Session-Ansicht. In dieser Session-Übersicht repräsentiert jede Clip-Launch-Taste je einen 5-Szenen-mal-8-Spuren-Block mit Clips, sodass die Matrix in der Übersicht insgesamt 125 Szenen und 64 Spuren darstellt. Drücken Sie eine Taste der Matrix, um auf den gewünschten Bereich der Session-Ansicht zu fokussieren. Wenn Sie z.B. die Shift-Taste halten und dann den Taster in der dritten Reihe der ersten Spalte drücken, fokussieren Sie auf die Szenen 11-15 und die Spuren 1-8.
SessionOverview.tif

Die Farbkodierung in der Session-Übersicht unterscheidet sich etwas von der normalen Kodierung:

-> Orange: zeigt den momentan ausgewählten Block mit Clips an, der in der Software von der Umrandung umgeben ist.
-> Grün: in diesem Block werden Clips abgespielt (unabhängig davon, ob der jeweilige Block mit Clips gerade ausgewählt ist oder nicht).
-> Rot: in diesem Block werden keine Clips abgespielt.
-> Keine Farbe: in diesem Block sind keine Spuren oder Szenen enthalten.

$Page Mixer-Bereich
$TargetName Mixer Section

Der Mixer-Bereich gibt Ihnen umfassende Steuerungsmöglichkeiten für Ihren Mix, wie z.B. Solo-schalten, Vorhören, Stummschalten, Einstellen von Panorama, Send-Pegeln und mehr. Die APC20 bietet acht Spur- und einen Masterfader, deren Zugriffsbereich sich in der Session-Übersicht noch erweitern lässt. Der /Cue-Level-Regler/ steuert die Vorhörlautstärke, sodass Sie einen Clip oder das Metronom vorhören können, ohne dass Ihr Publikum dies hört.
MixerSection.tif

Sie können diese acht Spuren innerhalb der Session-Ansicht verschieben. Drücken Sie die Right- oder Left-Taste, um sich schrittweise eine Spur weiterzubewegen. In der Session-Übersicht können Sie jeweils in Blöcken von acht Spuren weiter springen.

Die Tasten /Vol, Pan, Send A, Send B, Send C, User 1, User 2/ und /User 3/ bieten zusätzliche Steuerungsmöglichkeiten, indem die acht Fader der gewünschten Funktion neu zugeordnet werden. Wird z.B. Pan gewählt, können Sie mit den acht Fadern das  Spurpanorama einstellen.

Um die Fader-Funktion zu ändern, halten Sie einfach die Shift-Taste gedrückt und wählen den Parameter aus, den Sie steuern möchten. Die gewählte Parameter-Bank leuchtet dann rot auf, um zu zeigen, dass sie selektiert ist. Wenn Sie die Shift-Taste loslassen, bleiben die Fader der entsprechenden Funktion fest zugeordnet.

Die User-Bänke lassen Sie Lives /MIDI-Zuweisungs-Modus/ verwenden, um die Faderbelegung individuell anzupassen und zu bestimmen, welche Parameter jeweils von den Fader-Bänken gesteuert werden. Probieren Sie z.B. einmal aus, Ihren Fadern das Set-Tempo oder auch die globale Groove-Intensität zuzuweisen.

Der Master-Fader arbeitet in allen Modi immer als Master-Fader.

$Page Kombinations-Modus
$TargetName Combination Mode

Ihr volles Potential entfaltet die APC20-Bedienoberfläche, wenn Sie sie mit weiteren APC20 oder einer Akai Professional APC40 kombinieren.

Wird mehr als eine APC mit Ableton Live verbunden, arbeiten die Bedienoberflächen automatisch im /Kombinations-Modus./ Sie können insgesamt bis zu sechs APC20 oder APC40 kaskadieren. Das in den Voreinstellungen oben zuerst gewählte Gerät steuert die Spuren 1-8, die zweite gewählte Oberfläche die Spuren 9-16 und so weiter. Sind eine APC40 und eine APC20 gleichzeitig angeschlossen, dienen die Tasten der APC20 standardmäßig der Spuranwahl und die Fader der Lautstärkesteuerung, was den schnellen Zugriff auf einen erweiterten Bereich von Clips und Spuren ermöglicht.
