*Ableton Live Lite 6 Digidesign Edition* ofrece muchas de las funciones de Ableton Live 6 - la revolucionaria y varias veces galardonada solución de producción musical que le permite simultáneamente componer, grabar, remezclar, improvisar y editar espontáneamente sus ideas musicales en un entorno integrado de audio/MIDI. Esta edición de Live Lite ofrece:

-> 4 pistas de Audio y 4 pistas MIDI
-> Grabación a 32-bit
-> MIDI Remote Control
-> Soporte completo para ReWire
-> 2 Mono Ins y Outs
-> 4 Escenas en la Vista Session
-> ...y mucho más

Para aprender a usar Live, debería seguir las /lecciones/ incluidas. Las lecciones de Live son cortos tutoriales interactivos que ayudan a los principiantes a iniciarse y a los usuarios experimentados a sacarle más partido a Live. Puede accederse a ellas en cualquier momento eligiendo "Lecciones Tabla de contenidos" desde el menú Ayuda. Las siguientes lecciones le ayudarán a iniciarse:

-> $Link Grabación de Audio <Lesson:Builtin/1-Live Basics Category/1-Recording Audio>
-> $Link Reproducción de instrumentos software <Lesson:Builtin/1-Live Basics Category/2-Playing Software Instruments>
-> $Link Improvisación con Loops <Lesson:Builtin/1-Live Basics Category/3-Improvising with Loops>
-> $Link Creación de ritmos <Lesson:Builtin/1-Live Basics Category/5-Creating Beats>

Pulse sobre el botón /Página siguiente/ en l aparte inferior para ver una tabla comparativa de las características de esta edición de Live Lite con las de la versión completa.

/Live Lite 6 Digidesign Edition/ características:

-> 8 Pistas (pistas 4 Audio, 4 MIDI)
-> 2 Mono Ins, 2 Mono Outs
-> Grabación a 32-bit
-> MIDI Remote Control
-> MIDI Out para sintes hardware
-> Soporte completo para ReWire
-> 2 efectos de Audio y 2 MIDI por Live Set
-> 1 instrumento VST/AU y 1 efecto por Live Set
-> 4 Escenas en la Vista Session
-> 2 Envolventes de Clip por Live Set

La /versión completa de Live 6/ se amplía con respecto a estas características ofreciendo:

-> Pistas y escenas ilimitadas
-> Entradas y salidas ilimitadas
-> Efectos ilimitados
-> Soporte completo para VST
-> Soporte MP3, Ogg Vorbis, FLAC
-> Creación y edición de Racks de instrumentos y efectos
-> ...y mucho más


Puede explorar la versión completa de Live activando el $Link /Modo Demo/ <Lesson:Builtin/Demo Mode>, o actualizarse a la versión completa pulsando $Link aquí <http://www.ableton.com/upgrade-live-lite>.
