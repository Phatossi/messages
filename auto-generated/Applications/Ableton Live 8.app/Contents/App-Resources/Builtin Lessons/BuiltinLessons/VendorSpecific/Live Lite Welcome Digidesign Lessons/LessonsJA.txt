Ableton Live Lite 6 Digidesign Editionは、数々の賞を受賞した先進的音楽制作ソリューションであるAbleton Live 6の主要な機能が実装されています。作曲、録音、リミックス、アレンジ演奏、アイデアの編集がシームレスなオーディオ/MIDI環境において自由自在に行えます。このエディションのLive Liteの機能には、以下が含まれています。

-> 4つのオーディオトラックと4つのMIDIトラック
-> 完全32ビット録音
-> MIDIリモートコントロール
-> ReWireに完全対応
-> 2つのモノラル・インとアウト
-> 4つのセッションビュー・シーン
-> ...その他多数の機能を搭載しています。

Liveの使用方法について学ぶには、付属レッスンを読み進めていくとよいでしょう。Liveのレッスンは簡潔なインタラクティブ・チュートリアルで、初心者がLiveの使用法に慣れるためのヒントや、操作に詳しいユーザーがLiveをさらに使いこなすためのコツが学べるようになっています。レッスンは、「ヘルプ」メニューの「レッスン目次」からいつでもアクセスすることができます。Liveを始めて使用する場合には、以下のレッスンからはじめて見ましょう。

-> $Link オーディオを録音する <Lesson:Builtin/1-Live Basics Category/1-Recording Audio>
-> $Link ソフトウェアインストゥルメントの再生 <Lesson:Builtin/1-Live Basics Category/2-Playing Software Instruments>
-> $Link ループを使って即興アレンジを行う <Lesson:Builtin/1-Live Basics Category/3-Improvising with Loops>
-> $Link ビートの作成 <Lesson:Builtin/1-Live Basics Category/5-Creating Beats>

下の「次のページ」ボタンをクリックし、このエディションのLive Liteと、フルバージョンのプロフェッショナルな機能との相違を比較することができます。

$TargetName Live Lite Feature Comparison

Live Lite 6 Digidesign Editionの機能:

-> 8つのトラック（4つのオーディオトラック、4つのMIDIトラック）
-> 2つのモノラル・イン、２つのモノラル・アウト
-> 完全32ビット録音
-> MIDIリモートコントロール
-> ハードウェア・シンセサイザーへのMIDIアウト
-> ReWireに完全対応
-> Liveセット1つごとに、2つのオーディオエフェクトと2つのMIDIエフェクトを使用可能
-> Liveセット1つごとに、1つのVST/AUインストゥルメントと1つのエフェクトを使用可能
-> 4つのセッションビュー・シーン
-> Liveセット1つごとに、2つのクリップエンベロープを使用可能

フルバージョンのLive 6では、以下の機能を使用できます。

-> トラック数とシーン数は無制限
-> インとアウトは無制限
-> エフェクトは無制限
-> VSTに完全対応
-> MP3・Ogg Vorbis・FLACをサポート
-> インストゥルメントとエフェクトラックの作成と編集
-> ...その他多数の機能を搭載しています。


$Link デモモード <Lesson:Builtin/Demo Mode>をオンにしてフルバージョンのLiveを試用するか、 $Link ここ <http://www.ableton.com/upgrade-live-lite>をクリックしてフルバージョンにアップグレードすることができます。
