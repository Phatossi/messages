$Page Raccolta di File Esterni
$TargetName Collecting External Samples

Durante il loro sviluppo, i Progetti Live finiscono spesso col fare riferimento a file situati in svariate collocazioni del computer. Mentre questo non è un problema di per sé, i problemi possono invece sorgere se questi file vengono inavvertitamente spostati, cancellati o modificati da altre applicazioni.

Per ridurre il rischio di perdere dei file, il File Manager di Live può copiare tutti i file esterni di un tuo Progetto direttamente nella cartella del Progetto. Live considera "esterno" qualsiasi file che si trovi al di fuori della cartella del Progetto, cioè:

-> File collocati nella libreria di Live;
-> File collocati nelle cartelle di altri Progetti;
-> File collocati altrove nel tuo computer.

Lo stato di questi file viene mostrato nella sezione /File esterni/ del File Manager:
ExternalSamplesSection.tif

Puoi raccogliere tutti i file delle categorie sopra indicate, attivandone le rispettive opzioni /Sì/ e poi cliccando sul pulsante /Raccogli e Salva,/ posto ai piedi del File Manager:
CollectSaveButton.tif

Ci sono alcune cose che dovresti tenere presenti prima di raccogliere i file:

-> Strumenti multi-campionati e librerie spesso sono estremamente grandi e copiandoli aumenteranno notevolmente le dimensioni della cartella del Progetto. Prima di raccoglierli, controlla la dimensione dei file elencati per ciascuna categoria.
-> Se tu e i tuoi colleghi avete tutti lo stesso set di file (campioni, etc.) installato nelle vostre Librerie - ad esempio la Essential Instrument Collection di Ableton - non avrete bisogno di raccogliere questi file per condividere i vostri Progetti.
-> I campioni e gli strumenti software soggetti a licenza, come quelli della EIC, non verranno caricati sui computer che non dispongono di licenze valide per usarli.

Quando sei sicuro di voler raccogliere ogni file esterno a cui fa riferimento il Live Set corrente, puoi utilizzare anche il comando *Raccogli tutto e Salva* del menu File.
