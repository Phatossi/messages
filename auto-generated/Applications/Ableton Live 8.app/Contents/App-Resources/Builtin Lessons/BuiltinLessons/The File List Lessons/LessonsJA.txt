$Page ファイルリスト
$TargetName The File List

ファイルリストには、検査中のLiveセットで使用されるサンプルが表示されます。
FileList.tif

新規サンプルをリストにドロップし、サンプルをリスト内で再配置することができます。新規サンプルの長さが古いサンプルと同じかそれより長い場合、オーディオクリップはワープマーカーを維持します。ブラウズし、「ホットスワップ」ボタンを使ってサンプルを再配置することもできます。

サンプルの「編集」ボタンをクリックし、サンプルを外部サンプルエディタで開きます。編集スイッチがオンになっている間、サンプルはオフラインになります。オーディオクリップは、サンプルの長さが変更されない場合に限り、ワープマーカーを維持します。

「ロケーション」列は、$0のライブラリ、プロジェクトフォルダ、別の外部フォルダのどこにサンプルが置かれているかを示します。不明なサンプルもここに表示されます。

$0が不明なファイルを検索した場合、「候補」列にその検索結果も表示されます。
