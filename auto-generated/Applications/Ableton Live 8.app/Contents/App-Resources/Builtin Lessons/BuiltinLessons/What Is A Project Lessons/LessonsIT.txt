$TargetName What is a Project?

Un Progetto Live è una cartella che può essere usata per organizzare tutti i file correlati: Live Set, Clip, preset e altri tipi file.
ProjectInBrowser.tif

Supponi, ad esempio, di lavorare su un brano musicale: inizi con un Live Set vuoto; poi registri dell'audio e quindi crei nuovi file campione; poi trascini dei campioni da qualche raccolta; lungo il tuo percorso salvi varie versioni del Live Set in modo da poter tornare indietro e fare dei confronti.

Tutti i file correlati a questo brano musicale saranno collocati nella cartella del Progetto - e il /File Manager/ di Live ti fornirà gli strumenti necessari per gestirli.

I Live Set vengono sempre salvati all'interno di un Progetto Live, ma non devi preoccuparti di creare manualmente le cartelle di Progetto: se non esistono già, Live le crea automaticamente.
