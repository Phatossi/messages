$TargetName What is a Project?

Un projet Live est un dossier qui peut servir à organiser tous les Sets Live, clips, presets ou autres fichiers y ayant trait.
ProjectInBrowser.tif

Imaginez, par exemple, que vous travaillez sur un morceau de musique: vous partez d'un Set Live vide; vous enregistrez de l'audio et créez ainsi de nouveaux fichiers échantillons; vous déposez des échantillons venant de collections; vous sauvegardez différentes versions du Set Live en cours de travail pour pouvoir revenir en arrière et comparer.

Tous les fichiers relatifs à ce morceau de musique seront conservés dans le dossier projet - et le /gestionnaire de fichiers/ de $0 vous procurera les outils dont vous avez besoin pour les gérer.

Les Sets Live sont toujours sauvegardés dans un projet Live, mais vous n'avez pas à vous soucier de créer manuellement des dossiers de projet: Live les crée automatiquement s'ils n'existent pas déjà.
