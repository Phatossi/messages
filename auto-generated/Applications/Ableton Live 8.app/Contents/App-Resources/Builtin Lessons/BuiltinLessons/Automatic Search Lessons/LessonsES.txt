$Page Localización de muestras no encontradas
$TargetName Locating Missing Samples

El Administrador de archivos de $0 puede ayudarle a localizar cualquier muestra que falte en sus Live Sets.
MissingFilesSection.tif


Puede usar estos dos métodos:

-> $Link Reparación manual arrastrando y soltando <GotoPage:Manual Repair>
-> $Link Búsqueda automática <GotoPage:Automatic Search>


$Comment -----------------------------------------------------------------------
$Page Reparación manual
$TargetName Manual Repair

Para recomponer manualmente una referencia rota hacia una muestra, localice la muestra no encontrada en el Navegador de archivos:
MissingSampleInBrowser.tif

Arrastre el archivo desde el Navegador de archivos, y suéltelo en la entrada correspondiente de la lista de archivos extraviados en el Administrador de archivos:
LocatedMissingSample.tif

Tenga en cuenta que puede actualizar la referencia de archivo usando cualquier archivo de muestra que desee; no tiene porque ser el mismo archivo que faltaba.


Por favor pulse el botón /Página siguiente/ en la parte inferior para saber más acerca de la Búsqueda automática del Administrador de archivos.


$Comment -----------------------------------------------------------------------
$Page Búsqueda automática
$TargetName Automatic Search

La función Búsqueda automática del Administrador de archivos es a menudo al forma más sencilla de reparar referencias rotas a muestras:
AutomaticSearchSection.tif

Puede pulsar sobre el botón /Go/ para comenzar la búsqueda inmediatamente, o desplegar el triángulo a la izquierda para mostrar las funciones de búsqueda:

-> *Buscar carpeta* incluye una carpeta definida por el usuario en la búsqueda;
-> *Buscar proyecto* incluye la carpeta de proyecto del Set actual;
-> *Buscar Librería* incluye la Librería de $0;
-> *Buscar cerca de muestras ya econtradas* incluye las carpetas de todas las muestras previamente localizadas;
-> *Requerir coincidencias exactas* sólo devuelve resultados que coinciden exactamente con las muestras que faltan;
-> *Volver a reescanear carpetas totalmente* reconstruye completamente el índice de búsqueda de $0, encontrando cualquier muestra que se haya añadida o desplazada por otras aplicaciones.


Por cada muestra que falte, la Búsqueda automática puede encontrar cualqueir número de /candidatos,/ o posibles coincidencias:

-> Si no se encuentran candidatos, puede o bien reiniciar la búsqueda usando menos restricciones o localizar la muestra manualmente;
-> Cuando un candidato es encontrado, $0 acepta el candidato y considera resuelto el problema;
-> Si se encuentra varios candidatos, pulse sobre el botón /Hot-Swap/ de la entrada, que se muestra abajo; los candidatos se mostrarán en forma de lista en el Navegador de archivos:
HotSwapMissingSample.tif
